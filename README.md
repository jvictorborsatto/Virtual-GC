# Virtual GC

## Overview

Virtual GC is an interactive educational tool for gas chromatography designed to help students, instructors, and researchers explore chromatographic concepts through visualization and parameter manipulation.

The software allows users to investigate how chromatographic conditions influence retention, resolution, peak shape, efficiency, analysis time, and quantitative calibration without requiring access to laboratory instrumentation.

Virtual GC is intended for teaching and learning purposes and should not be used to predict real experimental results or develop analytical methods for laboratory applications.

---

## Main Features

### Method Optimization

Explore the influence of chromatographic parameters on separation performance:

#### Stationary Phase Selection

* Phase-5 OT (5% phenyl / 95% PDMS)
* Phase-WAX OT (polyethylene glycol)
* Phase-17 OT (50% phenyl / 50% PDMS)
* Phase-17 Packed

#### Column Parameters

* Column length
* Internal diameter
* Film thickness

#### Carrier Gas

* Helium (He)
* Hydrogen (H₂)
* Nitrogen (N₂)

#### Injection Conditions

* Splitless injection
* Adjustable split ratios

#### Temperature Programming

* Injector temperature
* Initial oven temperature
* Initial hold time
* Temperature ramp
* Final oven temperature
* Final hold time
* Detector temperature

#### Detector Selection

* Flame Ionization Detector (FID)
* Thermal Conductivity Detector (TCD)
* Electron Capture Detector (ECD)
* Mass Spectrometry (MS)

#### Signal Quality

* Adjustable noise
* Baseline drift
* Data acquisition density (points per minute)

---

### Chromatogram Customization

Create custom chromatograms by defining:

* Retention times
* Peak heights
* Peak widths
* Peak colors
* Signal quality settings
* Baseline characteristics

Useful for creating educational figures, exercises, exams, presentations, and teaching materials.

---

### Quantitative Analysis

Generate calibration datasets with:

* Multiple calibration levels
* Linear response models
* Quadratic response models
* Logarithmic response models
* Replicate generation
* Adjustable signal quality

Suitable for teaching:

* Calibration curves
* Peak integration concepts
* Quantitative chromatography
* Method validation fundamentals

---

### Visualization Tools

Available visualization options include:

* Peak labels
* Retention time annotations
* Peak height annotations
* FWHM annotations
* Resolution annotations
* Run comparison mode
* Overlay visualization
* Adjustable plotting window

---

### Export Options

Generated data can be exported as:

* PNG images
* CSV files
* ZIP archives

---

## Educational Purpose

Virtual GC was developed as a teaching and learning tool.

The software illustrates chromatographic trends and parameter relationships and is intended to support classroom activities, demonstrations, self-study, and introductory training in gas chromatography.

The generated chromatograms are educational representations and should not be interpreted as experimentally validated chromatographic predictions.

---

## System Requirements

* Modern web browser
* No installation required
* No external server required

Virtual GC operates locally as a standalone HTML application.
